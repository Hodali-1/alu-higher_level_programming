# Python - Classes and Objects

This project introduces Object-Oriented Programming (OOP) in Python:
classes, objects/instances, attributes, methods, `self`, `__init__`,
data abstraction/encapsulation, properties, and how Python looks up
attributes on an object versus its class.

## Requirements
- Editors: `vi`, `vim`, `emacs`
- Interpreted on Ubuntu 20.04 LTS, python3 (version 3.8.5)
- All files end with a new line
- First line of every file: `#!/usr/bin/python3`
- Code follows pycodestyle (version 2.7.*)
- All files are executable
- Every module, class, and function has a real, descriptive docstring
